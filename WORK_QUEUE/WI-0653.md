# WI-0653: Manager profile APIs and archetype mapping

**ID**: WI-0653
**Goal**: Add profile onboarding and CRUD APIs that map a fixed question set into the five FPL Sage archetypes plus deterministic draft/weekly constraints.

**Depends on**:

- WI-0652

**Scope**:
- `WORK_QUEUE/WI-0653.md`
- `cheddar-fpl-sage/backend/models/profile_api_models.py`

- `cheddar-fpl-sage/tests/test_profile_service.py`
- `cheddar-fpl-sage/tests/test_profiles_api.py`

**Out of scope**:
- Draft generation
- Screenshot upload/parsing
- Weekly dashboard rendering
- Learned personalization

**Acceptance**:
- `POST /api/v1/profiles` creates a manager profile from the five-question onboarding payload.
- `GET /api/v1/profiles/{user_id}` and `PATCH /api/v1/profiles/{user_id}` return and mutate persisted profile state.
- Archetypes are limited to Safe Template, Balanced Climber, Aggressive Hunter, Value/Flex Builder, and Set-and-Hold.
- Profile service emits deterministic constraint defaults for risk, bench preference, differentials, uncertainty tolerance, and early-transfer tolerance.
