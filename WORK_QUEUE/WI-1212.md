## Required Step 2 — Explicit Disposition

For each helper, choose exactly one disposition.

### A. Remove (preferred)

Conditions:

- Not reachable in runtime
- Not required for legacy adapter path

Requirements:

- Delete helper
- Remove all imports and usages
- Tests still pass

### B. Retain + Wire (only if proven necessary)

Conditions:

- Required for MLB legacy adapter or another active path

Requirements:

- Helper is invoked in transform-layer runtime
- Direct behavioral tests cover invocation and behavior
- No fallback decision logic is introduced
