# WI-0849: POTD Signal Engine — Candidate Generation, Scoring, Kelly Sizing

| Field | Value |
|---|---|
| **ID** | WI-0849 |
| **Goal** | Build the pure POTD signal engine that turns normalized odds into candidate picks across spreads, totals, and moneylines, scores them with the two-signal MVP, and sizes wagers with quarter-Kelly under a 20% cap. |
| **Scope** | `apps/worker/src/jobs/potd/signal-engine.js`, `apps/worker/src/jobs/potd/__tests__/signal-engine.test.js` |
| **Out of scope** | DB writes, Discord posting, scheduler registration, web/API code |
| **Acceptance** | (1) Candidate generation supports spreads (`HOME`/`AWAY`), totals (`OVER`/`UNDER`), and moneyline (`HOME`/`AWAY`) from the normalized odds structure. (2) `lineValue` and `marketConsensus` are the only active signals, weighted `0.625` and `0.375`; deferred signals are absent from scoring decisions. (3) The engine filters out candidates that cannot show a safe positive edge from consensus-fair probability versus locked implied probability. (4) Kelly sizing returns zero for non-positive edge and caps stake at 20% of bankroll. (5) Unit tests cover spread, total, and moneyline cases plus thresholding and Kelly edge cases. |
| **Owner agent** | unassigned |
| **Time window** | unassigned |
| **Coordination flag** | none |
| **Tests to run** | `npm --prefix apps/worker run test -- --runInBand src/jobs/potd/__tests__/signal-engine.test.js` |
| **Manual validation** | Feed representative NBA spread, NHL total, and MLB moneyline fixtures through the engine and confirm the chosen candidate, score breakdown, and wager sizing are plausible. |
