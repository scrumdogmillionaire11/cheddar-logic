## Background

A model can rank games correctly (high AUC) but be systematically overconfident or underconfident on its probability outputs. This miscalibration means the stated edge is not the realised edge. A model outputting `fairProb = 0.62` that wins at 0.52 is pricing in 10pp of fake edge.

Calibration must be a monitored, first-class metric — not something checked once during development and forgotten.

## Calibration Functions

```js
/**
 * Compute Expected Calibration Error for a set of predictions.
 * Bins predictions into deciles, compares avg prediction vs avg outcome per bin.
 *
 * @param {Array<{fair_prob: number, outcome: number}>} predictions
 * @param {number} [nBins=10]
 * @returns {{ ece: number, bins: Array<{bin: number, avg_pred: number, avg_outcome: number, n: number}> }}
 */
function computeECE(predictions, nBins = 10) {
  const bins = Array.from({ length: nBins }, (_, i) => ({
    bin: (i + 0.5) / nBins,
    preds: [],
    outcomes: [],
  }));

  for (const { fair_prob, outcome } of predictions) {
    const binIdx = Math.min(Math.floor(fair_prob * nBins), nBins - 1);
    bins[binIdx].preds.push(fair_prob);
    bins[binIdx].outcomes.push(outcome);
  }

  let ece = 0;
  const binStats = bins.map((b) => {
    if (b.preds.length === 0) return { bin: b.bin, avg_pred: null, avg_outcome: null, n: 0 };
    const avgPred = b.preds.reduce((s, v) => s + v, 0) / b.preds.length;
    const avgOutcome = b.outcomes.reduce((s, v) => s + v, 0) / b.outcomes.length;
    ece += (b.preds.length / predictions.length) * Math.abs(avgPred - avgOutcome);
    return { bin: b.bin, avg_pred: avgPred, avg_outcome: avgOutcome, n: b.preds.length };
  });

  return { ece, bins: binStats };
}

/**
 * Compute Brier score for a set of predictions.
 * Lower is better. Perfect = 0. Random 50/50 baseline = 0.25.
 */
function computeBrier(predictions) {
  if (predictions.length === 0) return null;
  const sum = predictions.reduce((s, { fair_prob, outcome }) => s + (fair_prob - outcome) ** 2, 0);
  return sum / predictions.length;
}
```

## Kill Switch Thresholds

| Market | ECE Kill Switch | Min Sample |
|---|---|---|
| NHL Total | > 0.06 | 50 |
| NBA Total | > 0.06 | 50 |
| MLB F5 Total | > 0.07 | 30 |
| Spread (any) | > 0.07 | 50 |
| ML (any) | > 0.08 | 50 |

When kill switch is active for a market: `isMarketCalibrationEnabled(market)` returns false; execution gate blocks cards for that market regardless of model status.
