# WI-1032: POTD edge-source contract — per-sport/market backing classification

**ID**: WI-1032
**Goal**: Define and enforce a machine-readable contract that classifies every sport+market combination as MODEL-backed or CONSENSUS_FALLBACK-backed for POTD candidate scoring, so cross-sport ranking never silently compares model-derived edges against market-shopping edges.

**Depends on**: WI-1029 (edgeSourceTag already stamped on candidates — this WI formalises the contract and adds enforcement), WI-1031 (vig accounting settled — informs which edge semantics count as "model")

**Background**:
After WI-1029, every `scoreCandidate` result carries `edgeSourceTag: 'MODEL' | 'CONSENSUS_FALLBACK'`. The problem being addressed here is that `selectTopPlays` currently ranks candidates from all sports together by `totalScore → edgePct`, regardless of whether `edgePct` was produced by a calibrated pitcher model (MLB ML) or by a simple best-book-vs-consensus arbitrage calculation (NBA spread, NHL totals). These edge values are not on the same scale and should not be ranked as if they were.

This WI introduces a static contract table and a cross-sport ranking guard that:

1. Declares the current backing status per sport+market.
2. Warns (not blocks) when a CONSENSUS_FALLBACK candidate outranks a MODEL candidate.
3. Provides a typed contract that WI-1030 and future model additions can update as each signal goes live.

**Scope**:

`resolveEdgeSourceContract(sport, marketType)` returns `'MODEL'`, `'CONSENSUS_FALLBACK'`, or `'UNKNOWN'` for unregistered combinations.

In `run_potd_engine.js`, after `selectTopPlaysFn` produces `fireableNominees`, if the top nominee has `edgeSourceTag === 'CONSENSUS_FALLBACK'` and any other nominee has `edgeSourceTag === 'MODEL'`, emit a structured warning log:

```json
{
  "type": "POTD_CROSS_RANKING_WARNING",
  "winner": { "sport": "...", "marketType": "...", "edgePct": ..., "totalScore": ... },
  "displaced_model_candidate": { "sport": "...", "marketType": "...", "edgePct": ..., "totalScore": ... },
  "note": "CONSENSUS_FALLBACK candidate ranked above MODEL candidate — verify edge scale parity"
}
```

This warning is informational only and does not block the play.

`edgeSourceTag` on each scored candidate must agree with `resolveEdgeSourceContract` for that sport+market. A mismatch indicates a scoring path bug. Add an assertion in the audit log path in `run_potd_engine.js` that logs `POTD_AUDIT_CONTRACT_MISMATCH` when the two disagree.

**Implementation notes**:

Add to `signal-engine.js` after `NOISE_FLOORS`:

```js
// Canonical per-sport/market edge source contract.
// MODEL = edge derived from a calibrated predictive model.
// CONSENSUS_FALLBACK = edge derived from market devigging (best book vs consensus).
// Update this table when a new model signal is wired in for a market.
const EDGE_SOURCE_CONTRACT = Object.freeze({
  MLB: Object.freeze({ MONEYLINE: 'MODEL', SPREAD: 'CONSENSUS_FALLBACK', TOTAL: 'CONSENSUS_FALLBACK' }),
  NHL: Object.freeze({ MONEYLINE: 'MODEL', SPREAD: 'CONSENSUS_FALLBACK', TOTAL: 'CONSENSUS_FALLBACK' }),
  NBA: Object.freeze({ MONEYLINE: 'CONSENSUS_FALLBACK', SPREAD: 'CONSENSUS_FALLBACK', TOTAL: 'CONSENSUS_FALLBACK' }),
  NFL: Object.freeze({ MONEYLINE: 'CONSENSUS_FALLBACK', SPREAD: 'CONSENSUS_FALLBACK', TOTAL: 'CONSENSUS_FALLBACK' }),
});

function resolveEdgeSourceContract(sport, marketType) {
  const sportKey = String(sport || '')
    .toUpperCase()
    .replace('BASEBALL_', '')
    .replace('ICEHOCKEY_', '')
    .replace('BASKETBALL_', '')
    .replace('AMERICANFOOTBALL_', '');
  const marketKey = String(marketType || '').toUpperCase();
  return EDGE_SOURCE_CONTRACT[sportKey]?.[marketKey] ?? 'UNKNOWN';
}
```

In `run_potd_engine.js`, add after the audit loop (before `viableCandidates`):

```js
// Cross-ranking warning: if the top fireable nominee is CONSENSUS_FALLBACK
// and any other nominee is MODEL, the ranking may be mixing incompatible edge scales.
if (POTD_AUDIT_LOG_ENABLED && fireableNominees.length > 1) {
  const topNominee = fireableNominees[0];
  if (topNominee?.edgeSourceTag === 'CONSENSUS_FALLBACK') {
    const displacedModel = fireableNominees.slice(1).find(n => n.edgeSourceTag === 'MODEL');
    if (displacedModel) {
      console.log(JSON.stringify({
        type: 'POTD_CROSS_RANKING_WARNING',
        winner: { sport: topNominee.sport, marketType: topNominee.marketType, edgePct: topNominee.edgePct, totalScore: topNominee.totalScore },
        displaced_model_candidate: { sport: displacedModel.sport, marketType: displacedModel.marketType, edgePct: displacedModel.edgePct, totalScore: displacedModel.totalScore },
        note: 'CONSENSUS_FALLBACK candidate ranked above MODEL candidate — verify edge scale parity',
      }));
    }
  }
}
```
