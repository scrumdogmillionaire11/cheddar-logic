# WI-1220: Confirm ESPN seriesStatus Field Path in NHL Playoff Model (R-05)

**ID**: WI-1220
**Goal**: Resolve the unconfirmed ESPN `seriesStatus` field path in `nhl-pace-model.js`, add a diagnostic log for when it is missing during playoff games, and add a test fixture asserting playoff feature extraction works with the confirmed path.

**Context**:

`apps/worker/src/models/nhl-pace-model.js:935` contains:

```
// TODO(WI-0970): seriesStatus field path not confirmed in ESPN payload.
// Verify against a live playoff snapshot before relying on these values.
const seriesStatus = oddsSnapshot?.raw_data?.seriesStatus ?? null;
```

The code is already defensive — if `seriesStatus` is null, all playoff regime fields (`playoff_series_game_num`, `playoff_elimination_pressure`) return null rather than wrong values. The risk is silent model degradation during playoffs: the playoff regime features do not contribute, causing the model to behave as if it is a regular-season game.

The NHL playoffs are active or approaching as of the current milestone. This must be resolved before the first playoff model run.

**Scope**:

- `apps/worker/src/scripts/probe-espn-series-status.js` (new) — one-shot diagnostic script to run against a live playoff odds snapshot
- `apps/worker/src/models/nhl-pace-model.js` — update `seriesStatus` field path once confirmed; add warning log when `isPlayoff=true` but `seriesStatus` is null; remove TODO comment
- `apps/worker/src/__tests__/run-nhl-model.test.js` — add test fixture with a synthetic NHL playoff odds snapshot asserting `playoff_series_game_num` is non-null when `seriesStatus` is present at the confirmed path

**How to confirm the field path (required first step)**:

Create and run `apps/worker/src/scripts/probe-espn-series-status.js` against a production DB snapshot (or dev DB with a copied playoff snapshot):

```js
'use strict';
require('dotenv').config();
const { getDatabase } = require('@cheddar-logic/data');
const db = getDatabase();
const rows = db.prepare(`
  SELECT game_id, raw_data FROM odds_snapshots
  WHERE sport = 'nhl' AND captured_at > datetime('now', '-48 hours')
  ORDER BY captured_at DESC LIMIT 10
`).all();
rows.forEach(r => {
  let raw = {};
  try { raw = JSON.parse(r.raw_data || '{}'); } catch {}
  const direct = raw?.seriesStatus;
  const nested = raw?.season?.seriesStatus;
  const event = raw?.event?.seriesStatus;
  console.log(r.game_id, {
    'raw_data.seriesStatus': direct ?? 'MISSING',
    'raw_data.season.seriesStatus': nested ?? 'MISSING',
    'raw_data.event.seriesStatus': event ?? 'MISSING',
  });
});
```
