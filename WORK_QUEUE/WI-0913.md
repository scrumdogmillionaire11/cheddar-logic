# WI-NEWS-0001: Cross-Sport News Signals Ingestion + Model Impact Contract

**ID**: WI-NEWS-0001
**Goal**: Add a deterministic cross-sport news ingestion and translation layer that writes structured context into model inputs, while adding NHL final-weeks/playoffs runtime weighting and gating without creating a separate playoff model.

**Depends on**: none

**Out of scope**:

- Building an LLM or free-text headline interpretation service
- Replacing sport-specific core models with new standalone playoff models
- UI redesign or dashboard feature work
- Broad refactors outside scheduler/news context wiring and NHL phase overlays

**Acceptance**:

- A normalized news object contract is available to model consumers (freshness, impact scores, tags, and normalized items with status/severity/targets) and is persisted in payload context used by model jobs.
- Scheduler includes cross-sport news polling stages that update normalized news signals before market/model execution windows, with critical-event fast refresh paths for impacted games only.
- NHL runtime keeps one engine path but applies bounded season-phase overlays (`REGULAR`, `FINAL_WEEKS`, `PLAYOFFS`) for recent-vs-season weighting, goalie-certainty gating strictness, and special-teams weight multipliers.
- Unconfirmed or contradictory high-impact news adjusts uncertainty/confidence and blocks executable outputs when required; confirmed critical news can invalidate stale decisions and trigger recomputation.
- Watchdog checks detect stale or unresolved critical news relative to decision timestamps and force conservative status (HOLD/PASS/BLOCKED) for affected outputs.
- Tests cover normalization, translation, playoff/final-weeks NHL overlay behavior, and stale-output invalidation guardrails.
