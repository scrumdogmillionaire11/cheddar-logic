# WI-0329: Move Hidden Errors to Collapsible Drawer

**ID**: WI-0329
**Goal**: Keep data error transparency while reducing clutter on the main board.

**Scope**:

- web/src/components/cards-page-client.tsx

**Out of scope**:

- Error detection logic
- Card filtering rules
- Backend data validation

**Acceptance**:

- Hidden errors block is collapsed by default in a details/summary drawer
- Summary text reads "N games excluded due to incomplete data"
- Expanded view shows matchup, game time, and missing inputs (up to a max list)
- Drawer is hidden entirely when hidden error count is zero
