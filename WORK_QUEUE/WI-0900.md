# WI-0900: Timestamp Integrity and Freshness Semantics Audit

**ID**: WI-0900
**Goal**: Standardize timestamp semantics for eligibility, surfacing, and settlement, and detect disagreements between data-time and write-time selection logic.

**Depends on**: WI-0899

- Timezone handling is documented with one canonical runtime policy.
- Conflict matrix documents behavior when timestamps disagree.
- At least three high-risk disagreement paths have explicit remediation tasks proposed.
