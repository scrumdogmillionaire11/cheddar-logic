# WI-1171: Remove Deprecated Execution-Gate No-Op Constants

**ID**: WI-1171
**Goal**: Delete the deprecated `VIG_COST_STANDARD` and `SLIPPAGE_ESTIMATE` no-op constants from the execution gate and remove any export surface that keeps them alive.

**Scope**:

- `apps/worker/src/jobs/execution-gate.js`
- `apps/worker/src/jobs/__tests__/execution-gate.test.js`

**Out of scope**:

- The module export surface no longer exposes deprecated vig/slippage constants.
- Narrow worker tests covering execution-gate behavior continue to pass without semantic drift.
