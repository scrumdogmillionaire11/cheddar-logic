## Background

The Market Pulse scanner (WI-0855) and API route (WI-0856) surface book-to-book divergences. This WI makes them visible in the web UI. The page is intentionally minimal — no model output, no picks, no card payloads. It is a pure odds-monitoring surface.

The page auto-refreshes every 5 minutes. Staleness is driven by `scannedAt` from the API response, not a client-side wall clock, so the badge is accurate even after a cached hit.

## Client Component Behavior

**Polling:**
- `setInterval` of 5 minutes triggers a fresh `fetch('/api/market-pulse?sport=...')`
- On mount: fetch immediately; clear interval on unmount

**Staleness badge:**
- Derived from `response.scannedAt` — display as "Last scanned X min ago"
- Recompute every 30 seconds via a separate `setInterval`
- Does NOT show stale state if `scannedAt` < 5 minutes ago

**Sport tabs:**
- `[All]` `[NBA]` `[MLB]` `[NHL]`
- Tab selection triggers `fetch('/api/market-pulse?sport=SPORT')` — full API call, not client-side filter
- Active tab highlighted

**"Show minor" toggle:**
- Collapsed by default (TRIGGER only displayed)
- When expanded: appends `&includeWatch=true` to the next fetch OR re-fetches with it
- WATCH-tier rows rendered at 50% opacity, smaller text (e.g. `text-sm opacity-50`)

**Display threshold for line gaps:**
- Client filters `lineGaps` to `delta >= 1.5` by default before rendering
- WATCH rows (delta 0.5–1.49) visible only when "Show minor" is active

**Copy constraints — zero forbidden terms:**
- Section header: "LINE DISCREPANCIES" (not "line picks")
- Row label: "OUTLIER" (not "bet this book")
- Odds gap subheader: "price varies across books"
- No use of: `bet`, `play`, `pick`, `recommend`, `wager`

## Acceptance Criteria

1. Navigating to `/market-pulse` renders the page without error (no TypeScript compile errors, no runtime crashes)
2. Sport tab `[NBA]` triggers a fetch to `/api/market-pulse?sport=NBA` and re-renders with filtered results
3. "Show minor" toggle re-fetches with `?includeWatch=true` and renders WATCH rows at reduced opacity
4. State 2 (empty) renders "Market is tight right now." when `lineGaps.length === 0 && oddsGaps.length === 0`
5. Staleness badge shows minutes since `scannedAt`, not a local timer
6. Auto-refresh polls every 5 minutes; component cleans up interval on unmount
7. Default line gap display threshold: only rows with `delta >= 1.5` rendered without "Show minor"
8. `grep -ri "bet\|pick\|recommend\|wager" web/src/app/market-pulse/ web/src/components/market-pulse/` → 0 matches in rendered text (strings, JSX, aria-labels)

6. Confirm staleness badge increments over 5 minutes and resets after auto-refresh.
