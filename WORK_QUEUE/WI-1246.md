# WI-1246: Calibrate High-End Slight Edge Promotion With Production Evidence

**ID**: WI-1246
**Goal**: Adjust high-end `LEAN -> PLAY` promotion thresholds or market coverage only where blocker audits and unified market-health data support it.

**Context**:

This is the first WI in this sequence that is allowed to tune promotion behavior, and it must remain market-specific. The audit and health-report phases come first so any change is tied to settled production evidence rather than anecdotal visibility.

**Scope**:
- `packages/models/src/decision-pipeline-v2-edge-config.js`
- `packages/models/src/__tests__/decision-pipeline-v2-promotion.test.js`
- `WORK_QUEUE/WI-1246.md`

**Out of scope**:
- Global `PLAY` threshold reductions
- Any change to canonical status taxonomy beyond promotion configuration
- Any change to POTD selection rules

**Acceptance**:
- Any promotion-threshold or promotion-scope change is explicitly justified by data from `WI-1244` and `WI-1245`.
- Changes are selective by sport/market and do not amount to a blanket threshold drop.
- Regression coverage proves each changed market behaves as intended and does not increase false positives outside the scoped cohort.
- Existing non-target markets remain unchanged.
