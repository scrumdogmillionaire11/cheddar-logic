# WI-0638: Remove NCAAM and Soccer sport filters from UI permanently

**ID**: WI-0638
**Goal**: Remove NCAAM and Soccer from all UI sport filter surfaces — filter pills, sport toggles, preset list, `TRACKED_SPORTS` constant, and NCAAM-specific render branches in `cards-page-client.tsx` — since neither sport will run live models again.

**Background**:
NCAAM is off-season indefinitely and Soccer is disabled on budget. Both are confirmed as non-returning model sports. Their sport filter buttons currently appear in the UI filter bar (filter-panel.tsx:154) and sport toggle (cards-page-client.tsx:53), creating dead filter options that return zero cards and confuse users. NCAAM also has a dedicated preset (`ncaam_ft_trend`) and a special-case render branch (`ncaam-ft-trend` card type check at cards-page-client.tsx:738, NCAAM spread check at 2509) that add code weight with no live utility.

The `Sport` type union entries (`'NCAAM'` | `'SOCCER'`) should be preserved — historical DB rows carry these values and API routes use them for filtering. Only the user-facing display surfaces are in scope.

**Scope**:
- `web/src/components/filter-panel.tsx` — remove `'NCAAM'` and `'SOCCER'` from `sportOptions` array (line 154)
- `web/src/components/cards-page-client.tsx`:
  - Remove `'NCAAM'` and `'SOCCER'` from `TRACKED_SPORTS` (line 53)
  - Remove the `ncaam-ft-trend` card-type branch (line ~738) and the `NCAAM spread` special-case (line ~2509); replace with no-op or remove entirely if the block has no other effect
- `web/src/lib/game-card/presets.ts` — remove the `ncaam_ft_trend` preset object (lines 115–124)
- `web/src/lib/game-card/filters.ts` — remove `'NCAAM'`, `'SOCCER'` from the three sport array literals (lines 128, 145, 158)

**Out of scope**:
- Changes to `Sport` type union in `game-card.ts` (historical DB rows still carry these values)
- Changes to API routes (`/api/games`, `/api/results`) — they still need to query/return NCAAM and Soccer rows for historical results
- Removing NCAAM/Soccer worker code (separate concern)
- NFL (tracked in WI-0639)

**Acceptance**:
- NCAAM and Soccer filter pills no longer appear in the cards UI filter bar.
- `TRACKED_SPORTS` does not include `'NCAAM'` or `'SOCCER'`.
- `ncaam_ft_trend` preset does not appear in the preset list.
- `npx tsc --noEmit --project web/tsconfig.json` passes.
