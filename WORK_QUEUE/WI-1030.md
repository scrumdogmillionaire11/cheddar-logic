# WI-1030: POTD — Wire NBA totals model signal into candidate scoring

**ID**: WI-1030
**Goal**: Replace the NBA POTD candidate's pure market-consensus edge with an actual model-derived edge for NBA totals markets, matching the pattern already used for MLB (pitcher model) and NHL (goalie composite). All candidates — MODEL and CONSENSUS_FALLBACK alike — must carry explicit source tagging and model metadata in the candidate payload so cross-sport ranking never mixes incompatible edge definitions silently.

**Depends on**: WI-1031 (vig accounting semantics must be settled first; WI-1030 must follow whichever `edgePct` convention WI-1031 establishes)

**Background**:
Currently, NBA POTD candidates for all market types fall through to the `scoreCandidate` fallback path in `signal-engine.js` which computes `edgePct = devigged_consensus_prob - implied_prob_at_best_price`. This is not a model edge — it measures "how much better is the best book than the consensus?" which is a line-shopping signal, not a predictive model signal. The NBA totals model (`run_nba_model.js`) produces `nba-totals-call` card payloads with a total projection that can be used as a signal source for NBA totals POTD candidates, exactly as NHL goalie output feeds NHL POTD candidates.

Spread and moneyline NBA POTD candidates remain on the consensus-fallback path for now; this WI scopes only totals since that is the only market the NBA model currently covers.

- In `scoreCandidate`, NBA TOTAL candidates with a valid `nbaSnapshot` signal use the model projection to compute `edgePct` instead of the consensus fallback:

  - `projection_source` stamped as `'NBA_TOTALS_MODEL'`.

**Implementation notes**:

`getLatestNbaModelOutput` pattern (mirrors `getLatestNhlModelOutput`):

```js
function getLatestNbaModelOutput(gameId) {
  const db = getDatabase();
  const row = db.prepare(`
    SELECT payload_data FROM card_payloads
    WHERE game_id = ? AND card_type = 'nba-totals-call'
    ORDER BY created_at DESC LIMIT 1
  `).get(gameId);
  if (!row) return null;
  const rd = JSON.parse(row.payload_data);
  const totalProjection =
    rd.projection_accuracy?.projection_raw ??
    rd.projection?.total ??
    rd.odds_context?.projection_comparison ??
    null;
  if (!Number.isFinite(totalProjection)) return null;
  return {
    totalProjection,
    projection_source: 'NBA_TOTALS_MODEL',
  };
}
```

`resolveNBAModelSignal` in `signal-engine.js`:

```js
function resolveNBAModelSignal(game) {
  const snap = game?.nbaSnapshot;
  if (!snap || !Number.isFinite(snap.totalProjection)) return null;
  return {
    totalProjection: snap.totalProjection,
    projection_source: snap.projection_source ?? 'NBA_TOTALS_MODEL',
  };
}
```

In `scoreCandidate`, add before the existing NHL block:

```js
const nbaSignal = candidate.nbaSnapshot ? resolveNBAModelSignal({ nbaSnapshot: candidate.nbaSnapshot }) : null;
const useNbaModelSignal =
  String(candidate.sport || '').toUpperCase() === 'NBA' &&
  candidate.marketType === 'TOTAL' &&
  nbaSignal !== null;
if (useNbaModelSignal) {
  const modelOverProb = clamp(0.5 + (nbaSignal.totalProjection - (candidate.consensusLine ?? candidate.line)) / 20, 0.05, 0.95);
  const modelSelectionProb = candidate.selection === 'OVER' ? modelOverProb : round(1 - modelOverProb, 6);
  const modelEdge = round(modelSelectionProb - impliedProb, 6);
  const totalScore = round((lineValue * 0.625) + (marketConsensus * 0.375), 6);
  return {
    ...candidate,
    lineValue,
    marketConsensus,
    totalScore,
    modelWinProb: round(modelSelectionProb, 6),
    impliedProb,
    edgePct: modelEdge,
    confidenceLabel: confidenceLabel(totalScore),
    scoreBreakdown: {
      lineValue,
      marketConsensus,
      model_win_prob: round(modelSelectionProb, 6),
      projection_source: nbaSignal.projection_source,
    },
    reasoning: buildReasoningString({
      selectionLabel: candidate.selectionLabel,
      price: candidate.price,
      edgePct: modelEdge,
      modelWinProb: round(modelSelectionProb, 6),
      lineValue,
      marketConsensus,
      marketType: candidate.marketType,
      projectionSource: nbaSignal.projection_source,
    }),
  };
}
```
