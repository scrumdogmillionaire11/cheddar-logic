**Goal**: Enforce single canonical sport values in card_payloads table to prevent future data quality issues and sport case mismatches.

- [packages/data/src/db.js](packages/data/src/db.js) — Verify all INSERT/UPDATE statements comply with constraint

- Sport case normalization implementation (WI-0315)
