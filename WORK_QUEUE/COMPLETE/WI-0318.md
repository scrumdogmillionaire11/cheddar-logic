### Phase 3: Consolidate Card Generation Logic (Medium-High Risk) — 4-8 hours

#### 3.1 Problem Statement

**Current state**: Each sport (NBA, NHL, NCAAM) has independent card generation with duplicated logic:

- [run_nba_model.js](apps/worker/src/jobs/run_nba_model.js#L230): `generateNBADriverCards()` + `generateNBAMarketCallCards()`
- [run_nhl_model.js](apps/worker/src/jobs/run_nhl_model.js#L240): `generateNHLDriverCards()` + `generateNHLMarketCallCards()`
- [run_ncaam_model.js](apps/worker/src/jobs/run_ncaam_model.js#L80): `generateNCAAMCards()`

**Duplicated patterns**:
- All create `card` objects with same structure (id, sport, cardType, payloadData)
- All validate with `validateCardPayload()`
- All call `insertCardPayload()`
- All attach runId with `attachRunId()`

**Why it's wrong**:
- Bug fix in one sport's card generation must be applied to all three
- Card structure changes require updates in three places
- Impossible to add a new card type or sport without duplicating entire card factory

#### 3.2 Implementation: Unified Card Factory

Create a sport-agnostic card factory in [packages/models/src/card-factory.js](packages/models/src/card-factory.js):

```javascript
/**
 * Build a standardized card payload
 * Used by: All driver models (NBA, NHL, NCAAM, etc.)
 * 
 * @param {object} input
 * @param {string} input.gameId
 * @param {string} input.sport - 'NBA' | 'NHL' | 'NCAAM'
 * @param {string} input.cardType - 'base-projection' | 'rest-advantage' | etc.
 * @param {string} input.cardTitle - Display title
 * @param {object} input.payloadData - Card-specific data
 * @param {string} input.createdAt - ISO timestamp
 * @returns {object} Card ready for insertCardPayload()
 */
function buildCard({
  gameId,
  sport,
  cardType,
  cardTitle,
  payloadData,
  createdAt = new Date().toISOString(),
  expiresAt = null,
  modelOutputIds = null,
  metadata = null,
}) {
  return {
    id: `card-${sport.toLowerCase()}-${gameId}-${uuidV4().slice(0, 8)}`,
    gameId,
    sport,
    cardType,
    cardTitle,
    createdAt,
    expiresAt,
    payloadData,
    modelOutputIds,
    metadata,
  };
}
```

Then simplify all model jobs:

```javascript
// run_nba_model.js (BEFORE)
const card = {
  id: `card-nba-${gameId}-${uuidV4().slice(0, 8)}`,
  gameId,
  sport: 'NBA',
  cardType: driver.type,
  cardTitle: `NBA ${driver.label}`,
  createdAt: now,
  expiresAt: null,
  payloadData: { ... },
  modelOutputIds: null,
};

// run_nba_model.js (AFTER)
const card = buildCard({
  gameId,
  sport: 'NBA',
  cardType: driver.type,
  cardTitle: `NBA ${driver.label}`,
  payloadData: { ... },
});
```
