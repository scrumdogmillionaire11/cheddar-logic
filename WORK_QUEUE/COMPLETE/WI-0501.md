### SQL / Log Evidence (queryable)

- Synthetic fallback measurability baseline (last 48h NHL SOG cards):

  ```sql
  SELECT COUNT(*) AS total_cards,
         SUM(CASE WHEN json_extract(payload_data, '$.decision.market_line_source')='synthetic_fallback' THEN 1 ELSE 0 END) AS synthetic_cards,
         SUM(CASE WHEN ABS(COALESCE(json_extract(payload_data, '$.drivers.opponent_factor'),1.0)-1.0)>0.0001 THEN 1 ELSE 0 END) AS non_neutral_opponent,
         SUM(CASE WHEN ABS(COALESCE(json_extract(payload_data, '$.drivers.pace_factor'),1.0)-1.0)>0.0001 THEN 1 ELSE 0 END) AS non_neutral_pace
  FROM card_payloads
  WHERE sport='nhl'
    AND card_type='nhl-player-shots'
    AND julianday(created_at)>=julianday('now','-48 hours');
  ```
