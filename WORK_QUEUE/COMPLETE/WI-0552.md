# WI-0552: Compute sigma empirically from game history

**ID**: WI-0552
**Goal**: Replace hardcoded sigma constants (NBA spread=12, total=14, etc.) with empirically computed rolling standard deviations from the games table so CDF-based edge math reflects actual season variance.

**Scope**:
- `packages/models/src/edge-calculator.js`
- `packages/data/src/db.js` (read-only query addition)
- `apps/worker/src/jobs/run_nba_model.js` (pass computed sigma at job start)
- `apps/worker/src/jobs/run_nhl_model.js` (same, if exists)
- `apps/worker/src/jobs/run_ncaam_model.js` (same, if exists)

**Out of scope**:
- Per-matchup or per-environment sigma (future enhancement)
- Changing threshold values — only sigma inputs to edge math
- MLB or NFL until those models are active

**Background**:
All spread and total edge calculations use `normCdf((T - mu) / sigma)`. Sigma is the single most load-bearing input: a 1-point error in NBA sigma shifts every edge calc by 0.3–0.8%, enough to flip LEAN→PASS or PASS→PLAY at threshold boundaries. Current constants (NBA spread sigma=12, total sigma=14) have no calibration lineage in the codebase.

**Acceptance**:
- A `computeSigmaFromHistory({ sport, marketType, db, windowGames = 60 })` function is exported from `edge-calculator.js` or a new `sigma-calibration.js` module in `packages/models/src/`.
- Function queries the games table for the last `windowGames` settled games for the given sport, computes standard deviation of `(home_score - away_score)` for `SPREAD` sigma and `(home_score + away_score)` for `TOTAL` sigma.
- Returns the computed sigma, or falls back to the existing hardcoded constant if fewer than 20 games are available (with a `sigma_source: 'fallback'` flag in the return object).
- Each model runner (NBA at minimum) calls `computeSigmaFromHistory` at job start and passes the result into `getSigmaDefaults` or directly into edge calculator calls.
- The `getSigmaDefaults` function accepts an optional override map so callers can inject computed sigmas without breaking the static fallback.
- A comment in `getSigmaDefaults` documents: current values, season they were set, and that they are fallbacks only.
