# WI-0521: Build deterministic Asian Handicap grading engine (incl. quarter splits)

**ID**: WI-0521
**Goal**: Implement a deterministic outcome engine for Asian Handicap that correctly grades whole, half, quarter, and zero (DNB) lines, including half-win/half-loss split semantics.

**Depends on**:

- `WORK_QUEUE/WI-0520.md`

**Scope**:

- `WORK_QUEUE/WI-0521.md`
- `apps/worker/src/models/soccer/asian-handicap-grader.js` (new)
- `apps/worker/src/models/soccer/__tests__/asian-handicap-grader.test.js` (new)
- `apps/worker/src/jobs/run_soccer_model.js`

**Out of scope**:

- De-vig and probability model calibration
- Card schema changes in `packages/data/src/validators/**`
- UI presentation updates

**Acceptance**:

- Engine accepts `team_goals`, `opponent_goals`, and `handicap` and derives outcome from `adjusted_diff = (team_goals - opponent_goals) + handicap`.
- Whole lines return `win|push|loss`; half lines return `win|loss`; zero line returns `win|push|loss`.
- Quarter lines split into two half-stakes with correct mapping (`±0.25`, `±0.75`) and combine to `full_win|half_win|half_loss|full_loss`.
- Invalid/malformed line values are rejected with explicit reason codes.
