## Background

The Market Pulse page (WI-0857) needs a server-side API endpoint that:
1. Fetches fresh odds snapshots from the DB (30-minute lookback window)
2. Runs both scanner passes ŌĆö line gaps first, then odds gaps on the clean games
3. Caches the result so that concurrent page loads don't each trigger a DB read + scan

One scan serves all users within the TTL window. The route is on-demand only ŌĆö no scheduler needed in v1. The page polls every 5 minutes, so the 4.5-minute TTL ensures at most 2 DB reads per 10-minute window under normal load.

## Response Shape

```typescript
// GET /api/market-pulse
// GET /api/market-pulse?sport=NBA
// GET /api/market-pulse?includeWatch=true
// GET /api/market-pulse?sport=MLB&includeWatch=true

interface MarketPulseResponse {
  scannedAt: string;          // ISO ŌĆö timestamp of the scan that produced this data
  lineGaps: LineGap[];        // sorted by delta desc; TRIGGER only unless ?includeWatch=true
  oddsGaps: OddsGap[];        // sorted by impliedEdgePct desc; TRIGGER only unless ?includeWatch=true
  meta: {
    gamesScanned: number;     // unique game_ids in the snapshot window
    booksScanned: number;     // unique books seen across all snapshots
    lineGapCount: number;     // total TRIGGER+WATCH before client filter
    oddsGapCount: number;     // total TRIGGER+WATCH before client filter
  };
}
```

`LineGap` and `OddsGap` shapes are defined in WI-0855.

## Cache Design

```typescript
// Server-side module-level cache (survives across requests in the same Node process)
const CACHE_TTL_MS = 4.5 * 60 * 1000; // 4 minutes 30 seconds
const LOOKBACK_MS  = 30 * 60 * 1000;  // 30-minute snapshot window

interface CacheEntry {
  payload: MarketPulseResponse;
  expiresAt: number;
}

// Keyed by sport string ('ALL' | 'NBA' | 'MLB' | 'NHL')
const cache = new Map<string, CacheEntry>();
```

Cache key is the normalized sport string only. `?includeWatch` is not part of the cache key ŌĆö the full TRIGGER+WATCH set is cached; the route trims to TRIGGER only when `includeWatch` is absent.

## Route Logic (pseudo-code)

```typescript
export async function GET(request: Request) {
  const { searchParams } = new URL(request.url);
  const sport       = searchParams.get('sport')?.toUpperCase() ?? 'ALL';
  const includeWatch = searchParams.get('includeWatch') === 'true';

  // Validate sport param
  const VALID_SPORTS = ['ALL', 'NBA', 'MLB', 'NHL'];
  if (!VALID_SPORTS.includes(sport)) {
    return Response.json({ error: 'Invalid sport' }, { status: 400 });
  }

  // Serve from cache if fresh
  const cached = cache.get(sport);
  if (cached && cached.expiresAt > Date.now()) {
    return serveResponse(cached.payload, includeWatch);
  }

  // Fetch snapshots
  const sinceUtc = new Date(Date.now() - LOOKBACK_MS).toISOString();
  const snapshots = sport === 'ALL'
    ? [
        ...getOddsSnapshots('NBA', sinceUtc),
        ...getOddsSnapshots('MLB', sinceUtc),
        ...getOddsSnapshots('NHL', sinceUtc),
      ]
    : getOddsSnapshots(sport, sinceUtc);

  // Scan line gaps
  const allLineGaps = scanLineDiscrepancies(snapshots);

  // Identify game_ids with at least one line gap ŌĆö exclude from odds scan
  const lineGapGameIds = new Set(allLineGaps.map(g => g.gameId));
  const cleanSnapshots = snapshots.filter(s => !lineGapGameIds.has(s.game_id));

  // Scan odds gaps on clean games only
  const allOddsGaps = scanOddsDiscrepancies(cleanSnapshots);

  // Collect meta
  const uniqueGames = new Set(snapshots.map(s => s.game_id));
  const uniqueBooks = extractUniqueBooks(snapshots); // parse raw_data.markets

  const payload: MarketPulseResponse = {
    scannedAt: new Date().toISOString(),
    lineGaps:  allLineGaps.sort((a, b) => b.delta - a.delta),
    oddsGaps:  allOddsGaps.sort((a, b) => b.impliedEdgePct - a.impliedEdgePct),
    meta: {
      gamesScanned: uniqueGames.size,
      booksScanned: uniqueBooks.size,
      lineGapCount: allLineGaps.length,
      oddsGapCount: allOddsGaps.length,
    },
  };

  cache.set(sport, { payload, expiresAt: Date.now() + CACHE_TTL_MS });
  return serveResponse(payload, includeWatch);
}

function serveResponse(payload: MarketPulseResponse, includeWatch: boolean) {
  if (includeWatch) {
    return Response.json(payload);
  }
  return Response.json({
    ...payload,
    lineGaps: payload.lineGaps.filter(g => g.tier === 'TRIGGER'),
    oddsGaps: payload.oddsGaps.filter(g => g.tier === 'TRIGGER'),
  });
}
```

## Acceptance Criteria

1. `curl localhost:3000/api/market-pulse` returns a 200 with `scannedAt`, `lineGaps`, `oddsGaps`, `meta` fields
2. `curl localhost:3000/api/market-pulse?sport=INVALID` returns 400
3. Two rapid successive calls within TTL window hit cache (verified by adding a log line or checking DB call count)
4. `curl localhost:3000/api/market-pulse` (no `includeWatch`) ŌåÆ all returned `lineGaps` and `oddsGaps` have `tier: 'TRIGGER'`
5. `curl localhost:3000/api/market-pulse?includeWatch=true` ŌåÆ includes `tier: 'WATCH'` items
6. `grep -r "bet\|play\|recommend\|pick" web/src/app/api/market-pulse/route.ts` (case-insensitive) ŌåÆ 0 matches (excluding this comment line)
7. `oddsGaps` contains no games that also appear in `lineGaps` (clean-game filter holds)
8. No writes to any DB table during the route handler
