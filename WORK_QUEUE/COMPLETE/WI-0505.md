**ID**: WI-0505

**Out of scope**:

- Altering Phase-1 classification bands or dead-zone rules
- Broad sportsbook integration changes outside NHL 1P gating path
- NBA/NCAAM/Soccer probability feature changes

**Acceptance**:

1. Phase-1 default remains unchanged: `fair_prob` for NHL 1P outputs stays `null` unless explicit Phase-2 gate is enabled.
2. Under gate + real 1P market line availability, fair probabilities are computed via normal CDF using `mu=final_1p_projection` and configured `sigma_1p` range.
3. If gate is enabled but 1P market line prerequisites are missing/unreliable, system fails safe to Phase-1 behavior (no priced recommendation leak).
4. Tests cover both gated-on and gated-off behavior, including dead-zone and goalie-uncertainty PASS interactions.
5. Docs and env flags clearly state that Phase-2 activation requires stable real 1P line supply.
