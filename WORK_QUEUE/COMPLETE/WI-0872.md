# WI-0872: Full-Game Total Model -- projectFullGameTotal()

**ID**: WI-0872
**Goal**: Add projectFullGameTotal() to mlb-model.js projecting full-game run totals by summing F5 (starter-driven) and late-innings (innings 6-9, bullpen-proxy) run segments.

**Scope**:
- apps/worker/src/models/mlb-model.js (new: projectFullGameTotal, projectLateInningsRuns)
- apps/worker/src/models/__tests__/mlb-model.test.js (6 new tests)
- apps/worker/src/jobs/run_mlb_model.js (wire projectFullGameTotal into computeMLBDriverCards as full_game_total card)

**Out of scope**: full-game ML (WI-0873), POTD wiring (WI-0874), DB migrations, umpire features, bullpen REST tracking

**Background**:
The pipeline reads full-game total lines (resolveMlbFullGameTotalContext) but produces no model projection.
data/mlb-research.md recommends modeling late innings via a bullpen proxy (team ERA x 4/9) not simply
scaling F5 by 9/5, because starter quality dominates inn 1-5 and bullpen dynamics dominate 6-9.

**Implementation notes**:
projectLateInningsRuns(offenseProfile, pitcherHandedness, opponentBullpenEra, environment) [plan name; spec had resolveLateInningsBullpenProxy — positional-param variant adopted for F5 consistency]:
  - bullpen_ra9 = context.bullpen_era ?? 4.3 (league avg)
  - lateMean = bullpen_ra9 * (4/9)
  - Apply resolveOffenseComposite() mult (capped [0.88, 1.14])
  - Apply same park/weather factors as F5
  - If bullpen_era null: degraded_input bullpen_era, use 4.3 default

projectFullGameTotal(homePitcher, awayPitcher, context):
  - Run projectTeamF5RunsAgainstStarter() for both sides (reuse)
  - Run projectLateInningsRuns() for both sides
  - fullGameMean = f5Home + lateHome + f5Away + lateAway
  - rangeWidth = sqrt(max(fullGameMean,0.1)) * 0.38 (wider than F5)
  - confidence: 7 base, +1 if both FULL_MODEL, -1 if bullpen_era missing
  - status_cap: PLAY if FULL_MODEL + bullpen present; LEAN if bullpen_era absent

**Acceptance**:
1. projectFullGameTotal() returns projected_total_mean, low, high, home_proj, away_proj
2. fullGameMean is greater than F5-only mean for the same pitchers
3. projection_source is FULL_MODEL when bullpen_era present; DEGRADED_MODEL when absent
4. Test: avg-starter + avg-bullpen game projects between 8.5-9.5 total
5. Test: elite-starter game projects lower total than avg-starter game
6. computeMLBDriverCards includes full_game_total card when full-game line is available
7. npm --prefix apps/worker test --no-coverage passes, zero regressions
