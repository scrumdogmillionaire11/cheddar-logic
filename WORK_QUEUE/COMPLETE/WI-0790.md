```

  Received: "PASS"
```

The swstr_pct null-guard in calculateProjectionK (mlb-model.js or mlb-k-input-classifier.js) no longer caps
status_cap to LEAN when swstr_pct is absent. Gate logic regressed or was removed.

### Failure 4 — mlb-k-statcast.test.js: null season_avg_velo missing_inputs


is no longer being pushed. Population logic regressed.
