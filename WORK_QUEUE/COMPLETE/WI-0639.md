# WI-0639: NFL UI seasonal gate — hide sport filter outside NFL season

**ID**: WI-0639
**Goal**: Hide the NFL sport filter pill and any NFL-specific presets in the cards UI when the current date is outside the NFL regular season + playoffs window (approximately September 1 – February 15). Show them automatically when NFL season resumes.

**Background**:
NFL appears in the sport filter today but no NFL model is running (off-season). Unlike NCAAM and Soccer (which are permanently removed per WI-0638), NFL will return as an active model each September. The current static `sportOptions` array includes NFL year-round, showing a dead filter option for ~7 months. A date-range gate in the filter removes it from the UI without requiring a deploy when the season starts/ends.

No NFL-specific presets exist in `presets.ts` today — this WI should future-proof by adding a guard at the preset registration site if any NFL presets are added later.

**Scope**:
- `web/src/components/filter-panel.tsx` — compute `isNflSeason` (true when current month is Sep–Dec or Jan–Feb) and exclude `'NFL'` from `sportOptions` when false
- `web/src/components/cards-page-client.tsx` — remove `'NFL'` from `TRACKED_SPORTS` when outside NFL season using the same `isNflSeason` helper, or exclude it from the initial filter state
- `web/src/lib/game-card/filters.ts` — remove `'NFL'` from the three static sport array literals; the filter bar drives sport selection, not these arrays (confirm the arrays aren't used as canonical sport lists elsewhere before removing)
- Extract `isNflSeason(): boolean` into `web/src/lib/game-card/season-gates.ts` (new, ~10 lines) so it can be unit tested and reused

**Out of scope**:
- Changes to `Sport` type union (`'NFL'` must remain — DB rows carry this value)
- Changes to API routes
- NFL-specific card rendering logic
- NCAAM and Soccer (WI-0638)
- Auto-detecting season boundaries from an external schedule API (hardcoded month range is sufficient)

**Acceptance**:
- NFL filter pill does not appear in the UI during off-season months (March–August).
- NFL filter pill appears automatically in September without a code change.
- `isNflSeason()` is unit-tested with at least 4 month boundary cases (Aug→false, Sep→true, Jan→true, Feb→true, Mar→false).
- `npx tsc --noEmit --project web/tsconfig.json` passes.
- `npm --prefix web run lint` passes.
- `npm --prefix web run test:ui:cards` passes.
