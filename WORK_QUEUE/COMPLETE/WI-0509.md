**Acceptance**:

1. NHL settlement prioritizes `api-web.nhle.com` free endpoints for game state/scores, with ESPN scoreboard fallback only when NHL API is unavailable.
2. NHL 1P settlement only locks when period-1 completion is verifiable (no pre-period-2 early settlement), and writes consistent first-period metadata for grading.
3. Player SOG settlement supports:
   - game-total SOG from NHL boxscore
   - 1P SOG by player from NHL play-by-play shot-on-goal event aggregation.
4. Settlement uses confirmation pass logic (pass-1/pass-2) and avoids auto-settling when values diverge between passes.
5. Coverage tests assert deterministic behavior for: NHL API happy path, ESPN fallback path, period-gating, and player-shots aggregation correctness.
6. No API key is required to run the settlement flow for NHL game/1P/player-shot results.
