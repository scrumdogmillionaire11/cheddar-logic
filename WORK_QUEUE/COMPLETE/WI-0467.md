# WI-0467: Automate NHL SOG Player ID Acquisition

**ID**: WI-0467
**Goal**: Auto-discover and maintain NHL SOG tracked player IDs from official NHL stats, then feed those IDs into the pull pipeline.

**Scope**:
- `WORK_QUEUE/WI-0467.md`

**Out of scope**:

**Acceptance**:
- `tracked_players` table exists with NHL SOG tracking fields and active-state support.
- `sync_nhl_sog_player_ids` pulls skater summary from NHL stats API (regular season), filters to `gamesPlayed >= 20`, ranks by shots per game, persists top 50, and deactivates stale tracked rows.
- `pull_nhl_player_shots` resolves IDs in order: DB tracked players -> env fallback -> exclude list.
- Scheduler queues `sync_nhl_sog_player_ids` at 04:00 ET with deterministic `jobKey`.
- Worker exposes `job:sync-nhl-sog-player-ids` script for manual runs.
- Docs/env reflect automated source with env fallback behavior.
