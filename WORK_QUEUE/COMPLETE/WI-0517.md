# WI-0517: Migrate GitHub Actions from Node 20 runtime to Node 24-compatible actions

**ID**: WI-0517
**Goal**: Remove Node.js 20 action-runtime deprecation risk by upgrading workflow actions to versions that support Node.js 24 before the June 2, 2026 cutoff.

**Scope**:

- `WORK_QUEUE/WI-0517.md`
- `.github/workflows/**`

**Out of scope**:

- Application runtime upgrades (Node version used by app/test jobs themselves)
- CI pipeline redesign unrelated to action-runtime deprecation
- Non-GitHub CI systems

**Acceptance**:

- No workflow in `.github/workflows/**` references deprecated action-runtime versions for:
  - `actions/checkout@v4`
  - `actions/setup-node@v4`
- Workflows use current Node 24-compatible major versions for those actions.
- If any required action has no Node 24-compatible replacement at execution time, add an explicit temporary mitigation in workflow env (`FORCE_JAVASCRIPT_ACTIONS_TO_NODE24=true` or `ACTIONS_ALLOW_USE_UNSECURE_NODE_VERSION=true`) with a TODO + removal condition.
- CI workflow YAML remains valid and existing triggers are unchanged.
