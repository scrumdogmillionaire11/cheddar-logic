# WI-0803: Remove dead npm scripts referencing non-existent job files

**ID**: WI-0803
**Goal**: Remove `package.json` scripts in `apps/worker` that reference job files that do not exist (`pull_schedule_ncaam.js`, `pull_soccer_player_props.js`, `pull_soccer_xg_stats.js`) so running any `npm run job:*` script does not silently or loudly fail due to a missing file.

**Scope**:

- `apps/worker/package.json`

**Out of scope**:

- Creating any of the missing job files
- Changes to any `.js` files
- `web/package.json`

**Acceptance**:

- The following scripts are removed from `apps/worker/package.json`:
  - `job:pull-schedule-ncaam`
  - `job:pull-soccer-player-props`
  - `job:pull-soccer-player-props:dry`
  - `job:pull-soccer-xg-stats`
- The `job:refresh-player-props` script is updated to remove the `&& node src/jobs/pull_soccer_player_props.js` segment (or removed entirely if it no longer makes sense)
- All remaining `job:*` scripts in `package.json` reference files that actually exist (`ls apps/worker/src/jobs/*.js` as verification)

**Owner agent**: unassigned
**Time window**: TBD
**Coordination flag**: solo

**Tests to run**:

- `npm --prefix apps/worker run test` (full suite must still pass — no job logic changes)

**Manual validation**:

- `npm --prefix apps/worker run job:pull-schedule-ncaam` → command not found (script removed)
- All other `job:*` scripts reference files confirmed to exist with `ls`

**Guard for WI closeout**:

- Zero references to non-existent files in `apps/worker/package.json`
- `job:refresh-player-props` (if kept) does not chain a non-existent file
