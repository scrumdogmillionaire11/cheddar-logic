# WI-0583 — V1 vs V2 SOG Projection Calibration Study

**ID:** WI-0583
**Goal:** Produce a documented calibration study comparing V1 (recency-decay blend) and V2 (rate-weighted Poisson) mu outputs on historic NHL SOG data, identify which model is more accurate, and set a formal accuracy gate before either model drives FIRE decisions alone.
**Severity:** MEDIUM
**Owner agent:** unassigned
**Status:** COMPLETE — Done in quick task 76 (2026-03-24, ea7b3b9)
**Time window:** —
**Coordination flag:** none

## Problem

V1 (recency-decay blend) and V2 (rate-weighted Poisson) produce different `mu` from the same player data with no reconciliation or accuracy validation. Currently V1 drives the bet direction and V2 is used only as an anomaly veto. Neither model has been back-tested against actual SOG outcomes to validate its accuracy at current parameter settings.

Without a calibration study:
- We cannot know which model to trust when they disagree.
- We cannot set a defensible `EDGE_MIN` threshold calibrated to actual hit rate.
- We may be over- or under-fitting recency weights.

## Scope

- New script: `scripts/calibrate_sog_v1_v2.js` — load all settled NHL player prop cards from `card_payloads` (last 90 days), compare `mu_v1`, `mu_v2`, and actual SOG outcome. Compute MAE and calibration (predicted vs actual over-rate at given lines).
- Output: `docs/runbooks/sog-v1-v2-calibration-YYYY-MM.md` with findings.
- No production code changes in this WI — observations only, used to inform WI-0554/future model updates.

## Out of Scope

- Changing model parameters.
- Changing any production job.
