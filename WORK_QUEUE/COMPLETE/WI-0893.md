# WI-0893: Add End-to-End Odds-to-Surface Drop Reason Ledger

**ID**: WI-0893
**Goal**: Add deterministic observability showing why odds-backed candidates do not surface, with layer-specific reason codes (worker gate, API filter, transform/filter).

**Depends on**: WI-0892

**Scope**:
- apps/worker/src/jobs/execution-gate.js

- Regression test verifies at least one candidate survives and at least one dropped candidate carries a reason code in synthetic fixture scenarios.
