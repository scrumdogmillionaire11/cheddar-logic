# WI-0847: Fix delta_pct ReferenceError in computeLineDelta — NBA line delta always null

| Field | Value |
|---|---|
| **ID** | WI-0847 |
| **Goal** | Fix a one-line camelCase/snake_case mismatch in `computeLineDelta` that causes every NBA model run to log `Failed to compute TOTAL/SPREAD line delta … delta_pct is not defined` and silently return `null` for `line_delta_pct` on every card payload. |
| **Scope** | `packages/data/src/db/odds.js` (line ~660) |
| **Out of scope** | Any other fields in the return object, NHL/MLB line delta paths, test schema changes |
| **Acceptance** | (1) `computeLineDelta` returns a numeric `delta_pct` when opener ≠ 0 and multiple snapshots exist — no longer `null`. (2) NBA model run no longer logs `Failed to compute … delta_pct is not defined`. (3) All tests pass. |
| **Owner agent** | unassigned |
| **Time window** | unassigned |
| **Coordination flag** | none |
| **Tests to run** | `npm test --workspace=packages/data -- --testPathPattern=odds --no-coverage --forceExit` and `npm --prefix apps/worker run test -- --no-coverage --forceExit --testPathPattern=run_nba_model` |
| **Manual validation** | Run `npm --prefix apps/worker run job:run-nba-model`; confirm no `Failed to compute TOTAL line delta` log lines appear |

## Root Cause

`packages/data/src/db/odds.js` line ~655–660:

```js
const delta = currentLine - openerLine;
const deltaPct =
  openerLine === 0 ? null : Number((delta / Math.abs(openerLine)).toFixed(4));

return {
  opener_line: openerLine,
  current_line: currentLine,
  delta: Number(delta.toFixed(4)),
  delta_pct,           // ← BUG: shorthand property references snake_case `delta_pct`
  snapshot_count: snapshotsWithLine.length,
};
```

The variable is named `deltaPct` (camelCase) but the return object uses `delta_pct,` as a shorthand, which is a ReferenceError. The function's `catch` in `buildMarketLineContext` (run_nba_model.js line ~316) swallows the error and returns `null`, so `line_delta_pct` is always `null` in every emitted card payload.

## Fix

One-line change in `packages/data/src/db/odds.js`:

```js
// Before
delta_pct,

// After
delta_pct: deltaPct,
```

## Notes

- The function is also imported by `run_nhl_model.js` if it uses line delta — check the NHL runner for the same `line_delta_pct` field after fix.
- Add a unit test asserting `computeLineDelta` returns a non-null `delta_pct` when two snapshots with different lines exist.
