## Background

The audit found that:
- GSaX (goals saved above expected) and raw SV% have r ≈ 0.65–0.75 for starters
- The pace model applies `applyGoalieAdj(savePct)` which reduces opponent scoring
- The cross-market NHL total driver uses `goalie_gsax` to produce a separate `goalieSignal` with weight 0.18

When both inputs are present for the same game, the shot-stopping quality of the goalie is counted twice in the full decision pipeline: once as a multiplier on raw goals, once as a signal driver. This creates a structural over-weighting of goalie quality in soft-scoring environments.

The fix: create a `resolveGoalieComposite(savePct, gsax)` function that combines both into a single normalized score. The pace model uses the composite to compute its adjustment factor; the cross-market driver reads the same composite.

## Composite Function

```js
// Reference values (2024-25 NHL season averages)
const LEAGUE_AVG_SAVE_PCT = 0.900;
const LEAGUE_AVG_GSAX_PER_GAME = 0.0;   // by definition: league sum = 0
const LEAGUE_GSAX_SD = 0.28;             // approx per-game SD for starters
const LEAGUE_SV_PCT_SD = 0.012;          // approx SD for starters

/**
 * Resolve unified goalie quality score.
 *
 * Returns a multiplier centred at 1.0:
 *   < 1.0 → better-than-average goalie (fewer goals allowed)
 *   > 1.0 → below-average goalie (more goals allowed)
 *
 * When both inputs are available: 70% GSaX z-score + 30% SV% z-score.
 * When only one is available: use that alone.
 * When neither: returns 1.0 (neutral).
 *
 * The output is calibrated so ±1 SD of composite = ±GOALIE_ADJ_SCALE * LEAGUE_GSAX_SD adjustment.
 *
 * @param {number|null} savePct
 * @param {number|null} gsax
 * @returns {{ factor: number, composite: number, source: 'FULL'|'SV_PCT_ONLY'|'GSAX_ONLY'|'NEUTRAL' }}
 */
function resolveGoalieComposite(savePct, gsax) {
  const hasSv = savePct !== null && Number.isFinite(savePct);
  const hasGsax = gsax !== null && Number.isFinite(gsax);

  if (!hasSv && !hasGsax) {
    return { factor: 1.0, composite: 0, source: 'NEUTRAL' };
  }

  let composite = 0;
  let source;

  if (hasSv && hasGsax) {
    const zSv = (savePct - LEAGUE_AVG_SAVE_PCT) / LEAGUE_SV_PCT_SD;
    const zGsax = gsax / LEAGUE_GSAX_SD;
    composite = 0.30 * zSv + 0.70 * zGsax;
    source = 'FULL';
  } else if (hasGsax) {
    composite = gsax / LEAGUE_GSAX_SD;
    source = 'GSAX_ONLY';
  } else {
    composite = (savePct - LEAGUE_AVG_SAVE_PCT) / LEAGUE_SV_PCT_SD;
    source = 'SV_PCT_ONLY';
  }

  // Positive composite = good goalie = REDUCES opponent goals (factor < 1.0)
  const rawFactor = 1.0 - composite * GOALIE_ADJ_SCALE * LEAGUE_GSAX_SD;
  const factor = clamp(rawFactor, GOALIE_ADJ_MIN, GOALIE_ADJ_MAX);

  return { factor, composite, source };
}
```

### Usage in `applyGoalieAdj`
```js
// Before: applyGoalieAdj(savePct, adjustmentTrust)
// After:  applyGoalieAdj(savePct, gsax, adjustmentTrust)

function applyGoalieAdj(savePct, gsax, adjustmentTrust) {
  if (adjustmentTrust === 'BLOCKED' || adjustmentTrust === 'NEUTRALIZED') {
    return { rawFactor: 1.0, appliedFactor: 1.0 };
  }
  const { factor: rawFactor } = resolveGoalieComposite(savePct, gsax);
  if (adjustmentTrust === 'DEGRADED') {
    return { rawFactor, appliedFactor: 1.0 + (rawFactor - 1.0) * 0.5 };
  }
  return { rawFactor, appliedFactor: rawFactor };
}
```

### Usage in `cross-market.js`
```js
// Replace goalieSignal computation:
const homeComposite = resolveGoalieComposite(
  raw?.espn_metrics?.home?.metrics?.goalieSavePct ?? null,
  goalieHomeGsax,
);
const awayComposite = resolveGoalieComposite(
  raw?.espn_metrics?.away?.metrics?.goalieSavePct ?? null,
  goalieAwayGsax,
);
// Composite centers at 0; positive = better goalie = UNDER pressure
const goalieSignal = clamp(
  -(homeComposite.composite + awayComposite.composite) / 2,
  -1, 1,
);
```
