# WI-0654: Draft sessions API, draft builder, and collaborative constraints

**ID**: WI-0654
**Goal**: Add draft-session CRUD plus draft generation that turns profile defaults and explicit user constraints into a primary build, a contrast build, and structured tradeoff notes.

**Acceptance**:
- `POST /api/v1/draft-sessions` and `PATCH /api/v1/draft-sessions/{id}` persist editable draft-session state.
- `POST /api/v1/draft-sessions/{id}/generate` returns one primary build and one contrast build.
- Supported collaborative constraints include `locked_players`, `banned_players`, `club_caps`, `bench_quality_target`, `premium_count_target`, `differential_slots_target`, `uncertainty_tolerance`, and `early_transfer_tolerance`.
- Supported intent phrases such as `keep Salah`, `fade Arsenal defense`, `make this safer`, `stronger bench`, and `one fun punt` translate into the same constraint model.
- Unsupported free-form text fails closed into guided-constraint feedback rather than pretending to understand.
