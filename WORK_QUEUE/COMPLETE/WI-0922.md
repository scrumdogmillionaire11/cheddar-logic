# WI-0922: Remove Broken Worker Job Script Entry Points

**ID**: WI-0922
**Goal**: Eliminate stale worker npm job scripts that reference non-existent files so operational commands reflect the real runnable surface.

**Depends on**: none

**Scope**:
- `apps/worker/package.json`
- `apps/worker/README.md`
- `scripts/refresh-data.sh`

**Out of scope**:
- Implementing or restoring removed sports pipelines (`soccer`, `ncaam`)
- Worker model algorithm changes
- DB/data-layer legacy compatibility cleanup

**Acceptance**:
- No `apps/worker/package.json` script references a missing `src/jobs/*.js` file.
- Composite scripts (`job:refresh-game-lines`, `job:dev-refresh`) no longer chain into missing jobs.
- Worker docs/scripts align with the updated script catalog and do not advertise missing jobs.
- A reproducible check command confirms script-target existence.
