## Implementation progress (2026-03-11)

- Added explicit Sports Reference fallback telemetry in `settle_game_results.js`:
  - logs each fallback attempt with ESPN miss reason context
  - logs explicit fallback no-match outcome reason
  - emits end-of-job fallback summary counters (attempts/matches/misses)
- Hardened miss reason composition to avoid malformed `undefined;...` reason strings.
- Preserved ESPN-first matching order and NCAAM-only fallback gate behavior.
- Added targeted NCAAM alias normalization to reduce unresolved results from team-name drift:
  - `LONG ISLAND UNIVERSITY` -> `LIU`
  - `GRAMBLING ST` -> `GRAMBLING`
- Expanded scoreboard date fetch window to include previous UTC day (`game day - 1, game day, game day + 1`) for date-boundary misses.
