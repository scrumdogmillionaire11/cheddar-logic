# WI-0947: MLB Totals Under-Bias Funnel Diagnosis and Tier Scaling

**ID**: WI-0947
**Goal**: Isolate and quantify MLB full-game totals under-bias by separating projection bias from gating and classification bias, then define bounded corrective adjustments.

**Depends on**: WI-0944

**Scope**:

- `apps/worker/src/models/mlb-model.js`
- `apps/worker/src/jobs/run_mlb_model.js`
- `apps/worker/src/jobs/__tests__/run_mlb_model.test.js`
- `apps/worker/src/models/__tests__/mlb-model.test.js`
- `WORK_QUEUE/WI-0947.md`

**Out of scope**:

- New DB schema/tables or migration changes
- Changes to NBA/NHL pipelines
- UI/web rendering contract changes
- Disabling existing guardrails without deterministic replacement

**Acceptance**:

- Directional funnel instrumentation is available for last N=200 MLB games with stage counts split by side (`OVER` vs `UNDER`) for: projection, edge, volatility, driver support, confidence, and execution.
- Funnel report includes per-side pre-gate and post-gate rates with: average model total, average market total, `% OVER`, `% UNDER`, and stage drop percentages.
- A suppressor attribution summary identifies top side-skewing gates with exact condition names and impact percentages.
- Volatility handling for full-game totals is reviewed for asymmetry and adjusted only if data shows overs are disproportionately suppressed by symmetric penalties.
- Bullpen and late-innings components used in full-game totals are audited for run-environment suppression and adjusted when they cause side-skewed flattening.
- Classification scaling for full-game totals is updated so materially larger run-delta edges do not collapse into the same tier as marginal edges; deterministic tests cover threshold behavior.
- Final output explicitly decomposes observed under-heavy board into shares attributable to projection mean bias, gate asymmetry, confidence compression, and tier flattening.
