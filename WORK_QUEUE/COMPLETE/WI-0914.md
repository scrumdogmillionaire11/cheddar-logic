**Out of scope**:

- Building a new standalone playoff model per sport
- UI redesign or adding new public pages
- New sportsbook/provider integrations
- Implementing soccer/tournament model runners (no current worker runtime)

**Acceptance**:

- A shared playoff overlay contract is available to all worker runners with deterministic fields for season phase, leverage context, volatility multipliers, and stricter execution thresholds.
- NHL playoff behavior applies stricter goalie-confirmation gating, 40-minute under-vs-full-game-volatility framing, special-teams amplification, and TOI concentration adjustments for eligible props.
- NBA playoff behavior applies rotation-shrink and star-minute concentration boosts, bench-prop suppression, lower blowout penalties, and tighter variance bands.
- NFL playoff behavior increases weather/conservatism weighting, raises market-edge thresholds, and concentrates confidence on primary usage players.
- MLB playoff behavior shifts weighting toward bullpen hierarchy and F5-vs-FG context, with weather suppression and lineup concentration impacts.
- Watchdog and scheduler honor playoff overlays and stale/high-impact context by preferring HOLD/PASS/BLOCKED when confidence is not actionable.
- Card payload validation accepts playoff overlay metadata and rejects malformed overlay contracts.
- Automated tests cover shared overlay normalization plus per-sport runner overlays and watchdog/scheduler interactions.
