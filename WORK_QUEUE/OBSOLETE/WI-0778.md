# WI-0778: Remove unrelated BMAD scaffolding and game-dev expansion packs from repo root

| Field              | Value |
|--------------------|-------|
| **ID**             | WI-0778 |

| **Priority**       | Low |
| **Owner agent**    | Unassigned |
| **Time window**    | TBD |
| **Coordination flag** | None — no application code references these directories |

---

## Problem

The repo root contains three directories that belong to a third-party BMAD AI workflow framework and have nothing to do with the cheddar-logic sports betting engine:

- `_bmad/` — BMAD core config, agents, tasks, and workflow templates (including brainstorming and party-mode workflows)
- `_bmad-output/` — generated BMAD output including a `deprecated/` sub-directory
- `web-bundles/` — BMAD expansion packs for Unity game dev, Godot game dev, creative writing, and infrastructure/devops — none applicable to this project

These inflate the repo size, confuse new contributors about what is application code, and may introduce false positives in grep/search operations across the codebase.

---

## Scope

| Path | Change |
|------|--------|
| `_bmad/` | Delete entire directory |
| `_bmad-output/` | Delete entire directory |
| `web-bundles/` | Delete entire directory |
| `.gitignore` | Add `_bmad/`, `_bmad-output/`, `web-bundles/` to prevent re-introduction |

## Out of Scope

- `pax-agents/` — this is a different, actively-used PAX agent tooling directory; do not touch
- `.claude/` — active Claude command/agent definitions; do not touch
- Any application source code
