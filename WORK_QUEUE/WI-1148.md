# WI-1148: Confidence Band Drilldown — Auditable Row List Per Band Pill

**ID**: WI-1148
**Goal**: Make each confidence band pill in the Projection Confidence Engine clickable, expanding an inline row-by-row game log (matchup, date, direction, outcome) so HIGH and STRONG records can be audited play by play.

**Scope**:

- `web/src/app/results/page.tsx`
- `web/src/components/results/ProjectionResultsTable.tsx`

**Out of scope**:

- API shape changes — all data needed already arrives via `/api/results/projection-settled` and `/api/results/projection-accuracy`
- Worker-side confidence normalization (covered by WI-1143 through WI-1146)
- Adding new confidence bands or changing thresholds
- Pagination or export features

**Acceptance**:

- Each band pill (LOW / WATCH / TRUST / STRONG / HIGH / MED) in the `by_confidence_band` breakdown is interactive.
- Clicking a pill expands an inline list of every graded directional row for that band: date, matchup, direction label, outcome (WIN/LOSS), and confidence score.
- HIGH and STRONG rows are visually distinguished from LOW/WATCH rows (color matches existing `confidenceBadgeClass` logic).
- The expanded list is collapsible (click the pill again to close).
- Rows with `weak_direction_flag = 1` are excluded from the drilldown list (consistent with how W/L counts are computed) and a count of excluded weak-direction rows is shown per band.
- If a band has 0 rows, its pill is non-interactive (cursor default, no expand).
- No regression in existing Confidence Engine stats display or `ProjectionResultsTable` row rendering.

**Manual validation**:

- Open `/results`, scroll to Projection Confidence Engine, click the `HIGH` pill — confirm a list of game rows appears showing matchup, date, direction, and WIN/LOSS outcome for each HIGH-band graded row.
- Click `STRONG` pill and verify the 162-win, 25-loss rows are individually listed.
- Click an already-open pill and confirm it collapses.
- Confirm LOW pill shows its rows and that they contain only strong-direction rows (weak_direction_flag = 0).
- Confirm a band pill with 0 rows is not clickable.
