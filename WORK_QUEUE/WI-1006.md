# WI-1006: Emit MLB F5 ML projection-only cards without odds dependency

**ID**: WI-1006
**Goal**: Ensure MLB F5 moneyline projections are emitted as projection-only cards even when dedicated F5 ML odds are unavailable.

**Depends on**: none

**Scope**:

- `WORK_QUEUE/WI-1006.md`
- `apps/worker/src/models/mlb-model.js`
- `apps/worker/src/jobs/run_mlb_model.js`
- `apps/worker/src/jobs/__tests__/run_mlb_model.test.js`
- `apps/worker/src/jobs/__tests__/run-mlb-model.dual-run.test.js`

**Out of scope**:

- Settlement/grading logic for MLB F5 ML outcomes
- Changes to bankroll sizing, staking, or execution thresholds
- Frontend route filtering and Game Props tab rendering logic

**Acceptance**:

- This WI alone is sufficient to ship projection generation and visibility for `mlb-f5-ml`; settlement hardening is tracked separately in `WI-1007`.
- `run_mlb_model` can emit `mlb-f5-ml` cards in `PROJECTION_ONLY` mode without requiring F5 ML odds prices.
- Emitted `mlb-f5-ml` payloads use deterministic `selection.side` values (`HOME` or `AWAY`) and never publish invalid PASS-side moneyline selections.
- Payload contract includes explicit projection-only markers (`basis` / `execution_status` / no-odds reason codes) plus stable market identity fields (`card_type=mlb-f5-ml`, `market_type=FIRST_5_INNINGS`, `recommended_bet_type=moneyline`) for downstream settlement and UI surfaces.
- Tests cover no-odds emission and guardrails for invalid moneyline selection payloads.

**Owner agent**: unassigned
**Time window**: 2026-04-17 -> 2026-04-21
**Coordination flag**: solo

**Tests to run**:

- `npm --prefix apps/worker run test -- src/jobs/__tests__/run_mlb_model.test.js --runInBand`
- `npm --prefix apps/worker run test -- src/jobs/__tests__/run-mlb-model.dual-run.test.js --runInBand`

**Manual validation**:

- Run MLB model in no-odds mode and confirm at least one `mlb-f5-ml` card is generated with `selection.side` of `HOME` or `AWAY`.
- Confirm emitted `mlb-f5-ml` cards are marked projection-only and not treated as executable odds-backed picks.
