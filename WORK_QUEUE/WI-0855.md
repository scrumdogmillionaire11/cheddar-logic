# WI-0855: Market Pulse Scanner Refactor — Split into `scanLineDiscrepancies()` and `scanOddsDiscrepancies()`

| Field | Value |
|---|---|
| **ID** | WI-0855 |
| **Goal** | Refactor `packages/models/src/mispricing-scanner.js` to export two focused functions alongside the existing `scanForMispricing()`. `scanLineDiscrepancies(snapshots, config)` detects books where the spread or total **number** diverges from consensus (existing LINE logic, now a first-class export). `scanOddsDiscrepancies(snapshots, config)` detects books where the **juice** differs on the same agreed line — covering spread price, total over/under price, and moneyline. | **Coordination flag** | None — `packages/models/` is not a serialized shared touchpoint |

## Background

The existing `scanForMispricing()` runs spread, total, and h2h scans in one pass and returns a flat `MispricingCandidate[]` mixing `edge_type: 'LINE'` and `edge_type: 'PRICE'` results. The Market Pulse API route (WI-0856) needs two distinct result sets:

1. **Line gaps** — the line number itself is disputed between books (meaningful signal; implies one book is off)
2. **Odds gaps** — all books agree on the number, but juice varies (weaker signal; represents shopping opportunity)

The spec requires `scanOddsDiscrepancies` to accept only **pre-filtered snapshots for games with no line gap** — the filtering logic lives in the API route caller (WI-0856), not in the scanner.

The existing ML thresholds (`nearEven.watch: 0.10`, `nearEven.trigger: 0.20`) are 10pp and 20pp implied probability differences, which are absurdly high and rarely fire in practice. A 20¢ odds difference on a near-even line (e.g. -105 vs -115) corresponds to ~2.3pp implied probability — the correct trigger threshold is `0.02`, not `0.20`.

## Output Shapes

### `LineGap`

```javascript
{
  gameId: string,           // snapshot.game_id
  sport: string,            // snapshot.sport
  homeTeam: string,         // from snapshot raw_data (if available, else null)
  awayTeam: string,         // from snapshot raw_data (if available, else null)
  commenceTime: string,     // from snapshot raw_data (if available, else null)
  market: 'spread' | 'total',
  outlierBook: string,      // the book with the divergent line
  outlierLine: number,      // the outlier line value
  consensusLine: number,    // median of remaining books
  delta: number,            // Math.abs(outlierLine - consensusLine)
  direction: 'home' | 'away' | 'over' | 'under',
  tier: 'TRIGGER' | 'WATCH',
  capturedAt: string        // snapshot.captured_at ISO string
}
```

### `OddsGap`

```javascript
{
  gameId: string,
  sport: string,
  homeTeam: string,
  awayTeam: string,
  commenceTime: string,
  market: 'spread' | 'total' | 'moneyline',
  line: number | null,       // agreed-upon line (null for moneyline)
  side: 'home' | 'away' | 'over' | 'under',
  bestBook: string,          // book with most favorable price for this side
  bestPrice: number,         // e.g. -105
  worstBook: string,
  worstPrice: number,        // e.g. -115
  impliedEdgePct: number,    // implied prob diff as decimal (e.g. 0.023 = 2.3%)
  tier: 'TRIGGER' | 'WATCH',
  capturedAt: string
}
```
