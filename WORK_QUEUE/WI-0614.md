# WI-0614: Rename decision-pipeline-v2.patch.js to decision-pipeline-v2-edge-config.js

**ID**: WI-0614
**Goal**: Rename `packages/models/src/decision-pipeline-v2.patch.js` to a name that accurately reflects it is an active production module, not a patch/diff file.

**Background**:
`decision-pipeline-v2.patch.js` is an active production module exporting real pipeline edge-config logic — not a patch or diff file. The `.patch.js` suffix causes engineers to avoid editing it (assuming it is generated) and may cause it to be silently skipped during updates. The correct name is `decision-pipeline-v2-edge-config.js` or similar.

**Scope**:
- Rename `packages/models/src/decision-pipeline-v2.patch.js` → `packages/models/src/decision-pipeline-v2-edge-config.js`
- Update all `require`/`import` references across the codebase to the new name
- Update any references in `package.json` exports if present

**Out of scope**:
- Changes to the module's logic or exports

**Acceptance**:
- `packages/models/src/decision-pipeline-v2.patch.js` no longer exists.
- `packages/models/src/decision-pipeline-v2-edge-config.js` exists with identical content.
- All callers import from the new path.
- `npm --prefix apps/worker test` passes.
