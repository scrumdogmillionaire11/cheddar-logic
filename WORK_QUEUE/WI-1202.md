**ID**: WI-1202
**Goal**: Refactor POTD engine to consume `DecisionOutcome` only, selecting strictly on `status='PLAY'`, eliminating fallback ghosts and drift from web/Discord.

**Scope**:
- `apps/worker/src/jobs/publish_potd_selection.js` (import canonical builder from @cheddar-logic/data, apply to decision_v2 list, filter outcomes for status='PLAY' only; remove all locally-defined decision_v2 → status mappers that are no longer called)

- Audit `apps/worker/src/utils/` for POTD-specific decision helpers that are no longer called after builder switch; delete easy hanging dead code (functions with no external callers, not exported publicly)

- **In scope (not deferred to WI-1203):** Removing "easy hanging" dead code — functions that become unused after builder switch, have no external callers, and are not exported. This reduces maintenance burden immediately.
