# WI-0561: Patch Next.js security advisories by upgrading to 16.1.7+

**ID**: WI-0561
**Goal**: Remove known security advisories affecting the current `next@16.1.6` production dependency.

**Scope**:
- `web/package.json`
- `web/package-lock.json`
- `web/` build/test scripts impacted by the upgrade (if needed)

**Out of scope**:
- Framework migrations beyond patch-level version bump
- Non-Next dependency upgrades
- Feature work in app routes/components

**Background**:
`npm --prefix web audit --omit=dev --json` reports advisories on `next` for the installed range `<16.1.7`, including:
- HTTP request smuggling in rewrites
- Unbounded next/image disk cache growth
- Unbounded postponed resume buffering (DoS risk)
- Null-origin CSRF bypass conditions

`web/package.json` currently pins `next` to `^16.1.6`.

**Acceptance**:
- `next` is upgraded to a patched version (`16.1.7` or newer secure patch).
- `npm --prefix web audit --omit=dev` reports zero known vulnerabilities for `next`.
- Web build and key regression tests pass on upgraded version.
