## Context

WI-0762 shipped `pull_vsin_splits.js` which fetches DraftKings public splits from VSIN using `source='DK'`. The existing `packages/adapters/src/vsin.js` already supports `source='CIRCA'` (Circa Sports sharp money) via the same HTML scraper — the `fetchSplitsHtml({ source: 'CIRCA' })` path is already implemented and tested.

**What Circa provides**: Circa Sports is a professional/sharp book. High handle % on a side at Circa relative to DK public ticket % indicates smart-money divergence — the classic "public one way, sharp money the other" signal.

**Divergence signal design**:
```
sharp_divergence = 'SHARP_VS_PUBLIC' when:
  |circa_handle_pct_home - dk_bets_pct_home| >= 20
  (sharp money on opposite side from public tickets)

sharp_divergence = 'SHARP_ALIGNED' when:
  |circa_handle_pct_home - dk_bets_pct_home| < 10

sharp_divergence = null when: either source is missing
```
