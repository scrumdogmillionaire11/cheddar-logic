## Implementation Summary

### Backfill Function

Added `backfillCardResultsSportCasing()` in `packages/data/src/db.js`:

- Counts rows with mixed-case sport values
- Updates all non-lowercase `card_results.sport` to lowercase via SQL `UPDATE`
- Returns `{affected: count, errors: null}` on success

### Guardrail Regression Tests

Four new tests in `packages/data/__tests__/card-payload-sport.test.js`:

1. **insertCardPayload test**: Verifies new card payloads and auto-enrolled results use lowercase sport (existing behavior).
2. **backfillCardResultsSportCasing test**: Inserts mixed-case sport values (NBA, Nhl, ncaam), runs backfill, verifies all normalized to lowercase (nba, nhl, ncaam).
3. **insertCardResult guardrail test**: Calls `insertCardResult()` with mixed-case input sport (NBA), verifies it persists as lowercase (nba).
4. **All rows regression test**: After backfill, verifies all `card_results.sport` values are lowercase.
