# WI-0738: Free MLB Pitcher-K Line Source Evaluation

**ID**: WI-0738
**Goal**: Evaluate quota-safe free line sourcing for MLB pitcher strikeout props and define a dedicated ingestion contract that can restore ODDS_BACKED comparisons without paid Odds API prop pulls.

**Scope**:
- `WORK_QUEUE/WI-0738.md`
- `docs/decisions/ADR-0009-mlb-pitcher-k-free-line-sourcing.md`
- `docs/runbooks/pitcher-ks-rollout.md`
- `apps/worker/src/jobs/run_mlb_model.js`
- `apps/worker/src/models/mlb-model.js`
- `packages/data/src/validators/card-payload.js`

**Out of scope**:
- Any paid Odds API pitcher-prop pull path
- Production deployment of brittle scraping without an ADR and backoff/rate-limit guardrails
- Changes to MLB F5 projection-only logic

**Acceptance**:
- ADR documents candidate free sources, source-of-truth preference, scraping/TOS/staleness risks, and a line schema for standard + alt K markets.
- One implementation path is selected, with DraftKings/FanDuel direct scraping as the preferred source and OddsTrader/OddsJam treated as secondary fallback candidates.
- Docs explicitly state there is no clean free structured MLB pitcher-K odds API and that runtime must stay projection-only until this WI ships.
