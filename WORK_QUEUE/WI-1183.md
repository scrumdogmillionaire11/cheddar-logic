## WI-1183

### 1. Calibration Guardrail (Blocker)

Raw delta must not automatically inherit blended-delta thresholds.

Add runtime instrumentation:

- Track `raw_delta`, `blended_delta`, `raw_side`, `legacy_side`, and surfaced status side-by-side.
- Emit shadow-eval records for NHL totals only.
- Evaluate on settled samples only (minimum 200 settled NHL totals over a rolling 14-day window).
- If sample size is below minimum, block promotion and continue shadow mode.
- If PLAY or LEAN rate increases beyond tolerance, block promotion and open a calibration WI.

Tolerance source of truth:

- `MAX_PLAY_RATE_DELTA_PCT`
- `MAX_LEAN_RATE_DELTA_PCT`
- `MAX_ECE_DELTA`

Acceptance:

```js
expect(sampleSize).toBeGreaterThanOrEqual(200);
expect(Math.abs(rawPlayRate - baselinePlayRate)).toBeLessThanOrEqual(MAX_PLAY_RATE_DELTA_PCT);
expect(Math.abs(rawLeanRate - baselineLeanRate)).toBeLessThanOrEqual(MAX_LEAN_RATE_DELTA_PCT);
expect(rawEce - baselineEce).toBeLessThanOrEqual(MAX_ECE_DELTA);
```

### 2. Severe Divergence Hard Stop With Explicit Precedence (Blocker)

`PASS_DIRECTION_MISMATCH` stops being the default terminal state, but severe contradiction still hard-stops weak raw edges.

Policy precedence (top to bottom, first match wins):

1. Neutral raw edge -> PASS_NO_DIRECTIONAL_EDGE
2. Opposite strong driver + weak raw edge -> PASS_SIGNAL_DIVERGENCE
3. Opposite driver + low-consensus raw edge -> PASS_LOW_CONSENSUS
4. Otherwise continue normal confidence/edge gates with divergence downgrade flag

Implementation skeleton:

```js
if (Math.abs(rawDelta) < RAW_DELTA_EPSILON) {
  return { status: 'PASS', reason: 'PASS_NO_DIRECTIONAL_EDGE' };
}

if (
  driverDirection &&
  driverDirection !== rawSide &&
  driverStrength >= STRONG_DRIVER_THRESHOLD &&
  Math.abs(rawDelta) <= RAW_DELTA_WEAK_EDGE_MAX
) {
  return { status: 'PASS', reason: 'PASS_SIGNAL_DIVERGENCE' };
}

if (
  driverDirection &&
  driverDirection !== rawSide &&
  Math.abs(rawDelta) < MIN_DIRECTIONAL_EDGE
) {
  return { status: 'PASS', reason: 'PASS_LOW_CONSENSUS' };
}

if (driverDirection && driverDirection !== rawSide) {
  confidence -= DIVERGENCE_PENALTY;
  flags.push('SIGNAL_DIVERGENCE');
}
```

Constant constraints:

- `RAW_DELTA_EPSILON <= RAW_DELTA_WEAK_EDGE_MAX < MIN_DIRECTIONAL_EDGE`
- `STRONG_DRIVER_THRESHOLD` must be defined from existing driver-strength scale

### 3. Epsilon / Neutral Zone (Warning -> Explicit)

Do not use literal zero for side derivation.

```js
const RAW_DELTA_EPSILON = 0.15;

if (Math.abs(rawDelta) < RAW_DELTA_EPSILON) {
  side = 'NONE';
  status = 'PASS_NO_DIRECTIONAL_EDGE';
}
```

Acceptance:

- `+0.01` does not become OVER.
- `-0.01` does not become UNDER.
- Neutral raw delta cannot surface as PLAY.

## Revised Runtime Rule

Direction authority:

- Primary: raw model delta
- Secondary: driver confirmation and divergence penalties
- Blended projection: pricing and display only

Divergence handling:

- Strong raw edge + opposite drivers -> downgrade and flag
- Weak raw edge + strong opposite drivers -> pass/block
- Neutral raw edge -> no play

Promotion guard:

- No production rollout without calibration guardrail pass on settled sample window.

Rollback:

- Feature flag required (`NHL_TOTALS_RAW_DELTA_AUTHORITY`).
- Immediate rollback path restores prior side-authority behavior without schema break.
