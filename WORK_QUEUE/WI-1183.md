# WI-1183: NHL Totals Directional Logic ‚Äî Raw Delta Authority, Divergence Guardrails, Reason Code Labels

**ID**: WI-1183
**Goal**: Replace `PASS_DIRECTION_MISMATCH` with explicit precedence-ordered pass reasons derived from raw delta authority and driver divergence policy; add epsilon/neutral zone; add four new reason code labels in shared and web maps; gate behind feature flag.

**Context**:

Original design spec condensed from WI-1183 (design doc). Shadow-eval calibration infrastructure is separated into WI-1224 (depends on this WI completing first).

**Driver signal source**:
- Cross-market path (run_nhl_model.js ~L2980): `driverDirection = totalDecision.best_candidate.side`, `driverScore = totalDecision.score` (0‚Äì1 numeric, cross-market score from `buildMarketDecision`)
- applyCanonicalNhlTotalsStatus path: no cross-market decision in context ‚Üí `driverDirection = null`, rules 2‚Äì3 of policy do not fire

**Constant values (authoritative)**:

```js
const RAW_DELTA_EPSILON       = 0.15;  // |delta| < this ‚Üí neutral, no play
const RAW_DELTA_WEAK_EDGE_MAX = 0.35;  // weak raw edge upper bound
const MIN_DIRECTIONAL_EDGE    = 0.50;  // matches existing slightEdge threshold
const STRONG_DRIVER_THRESHOLD = 0.65;  // above cross-market WATCH threshold (~0.5)
const DIVERGENCE_PENALTY_ABS  = 0.15;  // subtracted from absDelta when divergence flag fires
```

Constraint invariant: `RAW_DELTA_EPSILON (0.15) < RAW_DELTA_WEAK_EDGE_MAX (0.35) < MIN_DIRECTIONAL_EDGE (0.50)`

**Policy precedence (first match wins)**:

1. `|rawDelta| < RAW_DELTA_EPSILON` ‚Üí `PASS / PASS_NO_DIRECTIONAL_EDGE`
2. `driverDirection != null && driverDirection !== rawSide && driverScore >= STRONG_DRIVER_THRESHOLD && |rawDelta| <= RAW_DELTA_WEAK_EDGE_MAX` ‚Üí `PASS / PASS_SIGNAL_DIVERGENCE`
3. `driverDirection != null && driverDirection !== rawSide && |rawDelta| < MIN_DIRECTIONAL_EDGE` ‚Üí `PASS / PASS_LOW_CONSENSUS`
4. `driverDirection != null && driverDirection !== rawSide` ‚Üí continue with `absDelta -= DIVERGENCE_PENALTY_ABS`, `flags.push('SIGNAL_DIVERGENCE')`
5. Otherwise ‚Üí existing confidence/edge gates unchanged

`rawSide` derived from `rawDelta`: `delta > RAW_DELTA_EPSILON ‚Üí 'OVER'`, `delta < -RAW_DELTA_EPSILON ‚Üí 'UNDER'`, else `'NONE'`

`PASS_DIRECTION_MISMATCH` is retained in classifier for the legacy check path only (no new rows may produce it). Web/shared labels preserve its legacy text entry.
