# WI-1185: Canonical Decision Enforcement (Single Authority)

**Goal:**
Eliminate all read-time decision recomputation. Enforce canonical decision as the only source of truth across all runtime surfaces.

**Why:**
Current system allows multiple interpretations of the same row, creating contradictory outputs across APIs. This violates system integrity and produces undefined behavior. 

---

# NON-NEGOTIABLE RULE

```text
No runtime component may compute, infer, or modify decision state.

Decision = worker-produced canonical_decision ONLY
```

# NEW MODULE

## `packages/models/src/decision-authority.ts`

```ts
export function readCanonicalDecision(card) {
  if (!card?.decision_v2?.official_status) {
    return {
      status: "NON_ACTIONABLE",
      is_actionable: false,
      reason: "MISSING_CANONICAL_DECISION"
    };
  }

  return {
    status: card.decision_v2.official_status,
    is_actionable: card.decision_v2.is_actionable,
    classification: card.decision_v2.classification,
    reason: card.decision_v2.reason
  };
}
```

All consumers must import this. No duplication.
