**Acceptance**:
- `mlb-f5-ml` payloads include selected-side `win_probability`, `edge_pp`, `confidence_score`, and `confidence_band`.
- F5 ML is captured in `projection_accuracy_evals` as `MLB_F5_ML` using `synthetic_line=0.5` and `synthetic_rule=moneyline_baseline`.
- F5 ML settlement grades selected-side outcomes as `1`, `0`, or `0.5`, computes Brier score for non-push outcomes, and writes compatible `projection_proxy_evals` rows without fake placeholder edge values.
- `/results` API and UI expose F5 ML win probability, edge percentage points, confidence, tracking role, calibration bucket, Brier score, and expected outcome label.
- PASS rows remain calibration-only and are excluded from official pick stats.
- Legacy F5 ML rows missing prediction signal render as missing signal rather than synthetic `+0.50/-0.50` edge and `UNKNOWN` confidence.
