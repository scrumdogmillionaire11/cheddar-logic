# WI-1013: Resolve stale reason code alias confusion

**ID**: WI-1013
**Goal**: The reason codes `MARKET_DATA_STALE`, `STALE_MARKET_INPUT`, and `WATCHDOG_STALE_SNAPSHOT` are labelled as deprecated aliases in `packages/data/src/reason-codes.js` but are actively emitted and checked throughout the codebase.
