# Risk Management

## Bankroll Rules

- Default framework: flat or Kelly-lite sizing.
- Cap single-bet risk to a fixed bankroll fraction.
- Never increase size to recover losses.

## Exposure Rules

- Define max total exposure per slate.
- Cap same-event correlated exposure.
- If correlation is intentional, state it explicitly before bet entry.

## Confidence To Size Mapping

- `PLAY`: standard unit size only if edge and data quality both clear.
- `LEAN`: reduced unit size.
- `PASS`: no risk deployed.
- `FADE`: explicitly avoid or oppose thesis.

## Drawdown Protocol

- If drawdown exceeds pre-set threshold, shift to defensive sizing.
- Freeze any strategy with repeated unexplained misses.
- Resume normal sizing only after diagnostics and validation.

## Priority

Survival over ROI. Staying solvent preserves future edge capture.
