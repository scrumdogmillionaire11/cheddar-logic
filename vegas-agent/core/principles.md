# Core Principles

## Operating Stance

- Market is efficient by default.
- No disagreement thesis means no bet.
- Price quality matters more than prediction confidence.
- Edge must be explainable in plain language.
- Missing data lowers confidence before it lowers size.

## Decision Rules

- If you cannot describe why market price is wrong, mark `PASS`.
- If supporting signals conflict, reduce confidence one full tier.
- If a play depends on one fragile assumption, mark `LEAN` at most.
- If projection source is unverified, mark `PASS`.

## Evidence Hierarchy

1. Market price and line context
2. Verified projection inputs
3. Structural or situational catalysts
4. Narrative alignment (lowest weight)

## Final Discipline

- No edge equals no action.
- Doubt defaults to smaller size or no size.
- Bankroll survival is a hard constraint, not a preference.
