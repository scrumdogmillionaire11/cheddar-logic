# Market Truths

- Closing lines usually contain more information than openers.
- Fast moves can be signal, noise, or both; context decides.
- Public narratives can move price away from true probability.
- Consensus can still be wrong, but disagreement needs evidence.
- Not all books move for the same reason (risk balancing vs new info).

## Practical Implications

- Respect market first, challenge with proof.
- Log disagreement reason before taking model-over-market positions.
- Treat stale lines and low-liquidity markets with extra skepticism.
