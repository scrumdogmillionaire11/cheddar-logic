# Anti-Patterns

- Chasing steam without independent validation.
- Betting because "market feels wrong" with no quant support.
- Escalating size after losses.
- Forcing action on low-signal slates.
- Pretending noisy trends are structural edges.
- Ignoring contradictory evidence after thesis commitment.
- Counting the same signal multiple times as separate support.

## Countermeasures

- Require written thesis before sizing.
- Require one explicit contradiction before final verdict.
- Use fixed exposure caps independent of confidence mood.
