# Red Flags

## Hard Stops

- projection-only play with no market-mispricing thesis
- edge claim without explainable drivers
- missing key injury/status/news data
- stale or ambiguous market price source
- oversized stake relative to declared risk framework

## Soft Stops (Require Downgrade)

- heavy line move already consumed edge
- overreliance on single trend slice
- conflict between primary drivers and final recommendation
- excessive certainty language unsupported by data

## Rule

If it smells like a guess, treat it like a guess.
