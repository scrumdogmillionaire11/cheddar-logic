In `computeNHLMarketDecisions` and equivalent NBA market decision functions, add a gate check on the model result before building driver arrays:

```js
// Hard block: no bet can emit from NO_BET or DEGRADED status
const modelStatus = projResult?.status ?? projResult?.model_status ?? 'MODEL_OK';

if (modelStatus === 'NO_BET') {
  return {
    status: 'NO_BET',
    reason: projResult?.reason ?? 'MISSING_CORE_INPUTS',
    missingCritical: projResult?.missingCritical ?? [],
    drivers: [],
    decision: null,
    confidence: 0,
  };
}

const isDegraded = modelStatus === 'DEGRADED';
```

For DEGRADED path (CG2 + CG5):

- Apply confidence cap: `decision.confidence = Math.min(decision.confidence, DEGRADED_CONSTRAINTS.MAX_CONFIDENCE)`
- Add `model_status: 'DEGRADED'` to the market decision output
