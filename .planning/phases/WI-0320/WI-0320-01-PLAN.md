immediately after the mode comparison table. Content:

```markdown
### Bootstrap Requirement

`db-context.sh`, `start-scheduler.sh`, and `manage-scheduler.sh` **will not silently default** to
any DB path. They fail with an error if `CHEDDAR_DB_PATH` is not already set in the environment.

Before running any of those scripts directly, activate a mode first:

```bash
# Local mode (set once per terminal session)
source scripts/dev-mode-local.sh

# Snapshot mode
source scripts/dev-mode-snapshot.sh
```

Or set the variable explicitly:

```bash
export CHEDDAR_DB_PATH=/tmp/cheddar-logic/cheddar.db
```

`dev-mode-local.sh` and `dev-mode-snapshot.sh` are the only scripts authorized to set a default
`CHEDDAR_DB_PATH` value. All other runtime scripts inherit it or fail.
```

Do not alter the mode comparison table, one-time setup sections, or any other part of the file.
